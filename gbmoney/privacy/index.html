개인정보처리방침

최종 업데이트: 2026년 4월 15일
꼬박가계부(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

수집하는 정보

자동 수집 정보

  • 기기 ID (광고 게재 목적)
  • 충돌 데이터 및 진단 정보
  • 앱 사용 행동 분석 데이터 (Firebase Analytics)

수집하지 않는 정보

  • 이름, 이메일 등 개인 식별 정보 (로그인 기능 없음)
  • 가계부 데이터는 서버에 저장되지 않으며 기기 내에만 보관됩니다

정보 이용 목적

  • 앱 서비스 개선 및 오류 수정
  • 맞춤형 광고 제공 (AdMob)
  • 앱 사용 통계 분석

데이터 저장 방식

  • 모든 가계부 데이터는 기기 내에만 저장됩니다.
  • iCloud 동기화 이용 시 Apple의 iCloud 서버에 저장됩니다.
  • 구글 드라이브 백업 이용 시 Google Drive에 저장됩니다.
  • 각 클라우드 서비스의 개인정보처리방침을 별도로 확인하시기 바랍니다.

제3자 제공 및 SDK

광고

Google AdMob

광고 게재를 위해 기기 ID를 공유할 수 있습니다.

분석

Firebase Analytics

앱 사용 패턴 분석을 위해 익명화된 행동 데이터를 수집합니다.

데이터 보관 및 삭제

  • 앱 삭제 시 기기 내 모든 데이터가 함께 삭제됩니다.
  • iCloud 또는 구글 드라이브 백업 데이터는 각 서비스에서 직접 삭제하실 수 있습니다.

이용자의 권리

  • 광고 추적은 기기 설정에서 제한할 수 있습니다.
  • 앱 삭제를 통해 기기 내 모든 데이터를 삭제할 수 있습니다.

아동 개인정보

꼬박가계부는 전체 연령을 대상으로 하는 서비스입니다. 만 14세 미만의 경우 보호자의 동의 하에 이용을 권장하며, 아동의 개인정보는 보호자의 관리 하에 수집됩니다.
