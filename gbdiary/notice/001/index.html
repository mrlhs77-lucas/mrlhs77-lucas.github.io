Q
꼬박 클라우드가 뭔가요?
A
꼬박일기만의 클라우드 서버에 일기를 백업하고, 필요할 때 언제든 복원할 수 있는 서비스예요.
🤔 왜 만들었냐고요? 기존 구글 드라이브 백업은 2~3년치 데이터를 백업하는 데 너무 오랜 시간이 걸리고, 저장 공간이 부족하다는 문의가 많았어요. iCloud 동기화도 실시간 지원이 안 되고 사용자 iCloud 용량을 사용하다 보니 불편함이 있었고요. 꼬박 클라우드는 이런 문제들을 해결하기 위해 만들어졌어요.
Q
꼭 구독해야 하나요?
A
전혀 그렇지 않아요! 구독과 관계없이 아래 기능들은 모두 그대로 사용할 수 있어요.
일기는 항상 내 기기에 안전하게 저장돼요
iCloud 동기화 사용 가능(Apple 기기 한정)
구글 드라이브 백업 / 복원 사용 가능
PDF 내보내기 사용 가능
꼬박 클라우드는 더 빠르고 편리한 백업을 원하는 분들을 위한 선택지예요 😊
Q
갑자기 로그인이 생긴 이유가 있나요?
A
꼬박 클라우드 구독 서비스를 위해 로그인 기능이 추가됐어요.
✅
로그인은 선택사항이에요. 로그인 없이도 기존처럼 앱을 사용할 수 있어요
✅
꼬박 클라우드를 이용하려면 로그인이 필요해요
Q
구독하면 어떤 점이 좋은가요?
A
⚡
구글 드라이브보다 훨씬 빠른 백업 & 복원
💾
내 iCloud / 구글 드라이브 용량을 사용하지 않아요
🔒
꼬박 클라우드 서버에 안전하게 보관돼요
📱
새 기기로 바꿔도 빠르게 복원할 수 있어요
🐣 꼬박 Team의 한마디 소중한 일기는 항상 여러분 기기에 저장돼요. 꼬박 클라우드는 더 안심하고 싶은 분들을 위한 선택이에요. 앞으로도 더 좋은 꼬박일기가 되도록 노력할게요 💛
